# Chrome Extention - AI Korean Translator

AI를 이용하여 선택한 문장이나 현재 화면에 보이는 주요 텍스트를 한국어로 번역하는 크롬 확장입니다.

## 기능

- 선택한 문장만 번역
- 현재 화면에 보이는 주요 텍스트를 한 번에 번역
- 화면 번역 후 원문 복원
- 설정에 따라 문장 선택 시 `번역` 버튼 표시
- 또는 우클릭 메뉴로 번역
- 단축키 `Alt+Z`로 현재 선택 문장 번역
- 선택 문장 아래 말풍선으로 번역 결과 표시
- 말풍선에서 번역 결과 복사 또는 닫기
- 최근 번역 결과 캐시로 같은 문장 재번역 속도 개선
- 번역 톤 선택
- 번역 말풍선 글꼴 선택
- OpenAI API 키와 모델명을 옵션 페이지에 저장

## 파일 구조

- `manifest.json`: 확장 설정
- `content.js`: 현재 페이지의 선택 텍스트 감지
- `background.js`: OpenAI API 호출
- `popup.html`, `popup.js`: 번역 UI
- `options.html`, `options.js`: API 키 설정 UI
- `styles.css`: 팝업/설정 화면 스타일

## 설치 방법

1. 크롬에서 `chrome://extensions`를 엽니다.
2. 우측 상단의 `개발자 모드`를 켭니다.
3. 왼쪽 상단의 `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 다운로드한 폴더를 선택합니다.

## 사용 방법

1. 확장 옵션 페이지에서 OpenAI API 키를 저장합니다.
2. 번역하고 싶은 문장을 웹페이지에서 선택합니다.
3. 선택 문장 근처에 나타나는 `번역` 버튼을 누르거나 우클릭 메뉴 또는 `Alt+Z`를 사용합니다.
4. 선택 문장 근처의 말풍선에서 번역 결과를 확인합니다.
5. 현재 화면에 보이는 부분을 번역하려면 팝업의 `보이는 부분 번역` 버튼을 누릅니다.
6. 화면 번역을 되돌리려면 팝업이나 페이지 상단 알림의 `원문 복원` 버튼을 누릅니다.

단축키는 `chrome://extensions/shortcuts`에서 변경할 수 있습니다.
